# VedaAI Quick Start Guide

## Navigation Map

```
VedaAI Platform
├── HOME (/home) - Dashboard with stats & quick actions
├── ASSIGNMENTS (/assignments) - Manage all assignments
├── CREATE ASSIGNMENT (/create-assignment) - Multi-step wizard
├── LIBRARY (/library) - Browse resources & templates
└── AI TOOLKIT (/ai-toolkit) - AI-powered tools
```

## Page Summary

| Page | URL | Purpose | Key Features |
|------|-----|---------|--------------|
| **Home** | `/home` | Dashboard & Overview | Stats, Quick actions, Recent assignments |
| **Assignments** | `/assignments` | Manage Assignments | Grid view, Search, Filter, Create button |
| **Create Assignment** | `/create-assignment` | Build Assignments | 4-step wizard, Question config, File upload |
| **Library** | `/library` | Browse Resources | Search, Filter, Preview, Download, Save |
| **AI Toolkit** | `/ai-toolkit` | AI Tools | 6 tools, Usage stats, How-to guide |

## Quick Access (Desktop)

- **Home** → Click "Home" in sidebar
- **Assignments** → Click "Assignments" in sidebar  
- **Create Assignment** → Click "+ Create Assignment" button
- **Library** → Click "My Library" in sidebar
- **AI Toolkit** → Click "AI Teacher's Toolkit" in sidebar

## Quick Access (Mobile)

- **Home** → Bottom nav (house icon)
- **Assignments** → Bottom nav (folder icon)
- **Create Assignment** → "+" floating button
- **Library** → Bottom nav (clock icon)
- **AI Toolkit** → Bottom nav (spark icon)

## Color Coding

- **Orange** (#F97316) - Primary CTAs and highlights
- **Blue** - Secondary stats and metrics
- **Green** - Success indicators
- **Amber** - Warnings or pending items
- **Purple** - Tertiary information
